# 🚀 Portfolio Web Interactivo

Este es un repositorio donde daré forma a mi **portfolio personal** usando **HTML**, **CSS** y **JavaScript**. Es una página web interactiva que me presenta, describe mis habilidades e historia, con navegación mediante flechas hacia diferentes mini-proyectos que demuestran funcionalidades útiles en el mercado.

---

## 🧭 Estructura General del Proyecto

### 1. **Página de Inicio (`index.html`)**
- Foto o avatar, nombre y breve presentación personal
- Sección "Sobre mí": historia, formación, intereses
- Lista de **habilidades** (lenguajes, frameworks, soft-skills)
- Flechas izquierda y derecha para navegar entre proyectos
- Diseño limpio y atractivo con CSS moderno

### 2. **Navegación con Flechas**
- Elementos interactivos (botones o enlaces) posicionados en los laterales
- Animaciones suaves al pasar el ratón
- Navegan a diferentes páginas HTML que contienen mini-proyectos
- Posibilidad de crear un slider visual con transiciones

### 3. **Estructura de Carpetas** (Planeada)
```
Portfolio/
├── index.html           # Página de inicio
├── style.scss/
│   └── styles.css       # Estilos generales
├── SCRIPTS/
│   └── proyecto1.js        # Lógica de navegación y interactividad
│   ├── proyecto2.js
│   ├── proyecto3.js
│   └── ...
├── PROYECTOS/               # Páginas de proyectos
│   ├── proyecto1.html
│   ├── proyecto2.html
│   └── ...
└── README.md            # Este archivo
```

## 💡 Ideas para las Páginas de Proyectos

Cada página mostrará una funcionalidad específica y buscará demostrar habilidades técnicas reales:

### 1. **App del Tiempo (Weather App)** 🌤️
- Consume una API pública (OpenWeatherMap)
- Muestra temperatura, iconos, pronóstico simple
- **Demuestra:** consumo de APIs, manejo de promesas, DOM dinámico

### 2. **Calculadora Interactiva** 🧮
- Operaciones básicas y avanzadas
- Diseño responsivo con CSS Grid/Flexbox
- Validación de entrada y manejo de errores
- **Demuestra:** lógica con JS, UI/UX responsiva

### 3. **Lista de Tareas (To-Do List)** ✅
- Añadir, editar, eliminar y completar ítems
- Guardado en `localStorage` para persistencia
- Filtros por estado (activos, completados)
- **Demuestra:** manipulación del DOM, almacenamiento local, interactividad

### 4. **Página de Producto (E-commerce)** 🛒
- Galería de imágenes con lightbox
- Descripción de producto, precio, atributos
- Simulación de carrito de compras dinámico
- **Demuestra:** manejo de datos, eventos, CSS avanzado

### 5. **Cronómetro/Temporizador** ⏱️
- Contar hacia adelante o regresiva
- Botones de control (iniciar, pausar, reiniciar)
- Animaciones visuales de progreso
- **Demuestra:** `setInterval`, control de estado, UI clara

### 6. **Juego Sencillo** 🎮
- Tic-Tac-Toe, juego de memoria o Piedra-Papel-Tijera
- Lógica de turnos, puntuación, reinicio
- Interfaz clara y retroalimentación visual
- **Demuestra:** lógica compleja, manejo de estado, UX interactiva

### 7. **Formulario Dinámico con Validación** 📋
- Campos de registro/contacto con validación en vivo
- Mensajes de error personalizados
- Confirmación de envío (simulada o con API)
- **Demuestra:** validación, expresiones regulares, UX mejorada

### 8. **Galería de Imágenes con Filtro** 🖼️
- Categorías o búsqueda dinámica
- Filtrado en tiempo real
- Lightbox para ver imágenes en grande
- **Demuestra:** filtrado de datos, CSS avanzado, responsividad

## 📝 Objetivo General

Este sitio es mi portfolio personal interactivo. La página principal me presenta y enumera mis habilidades profesionales. Las flechas laterales permiten desplazarse a diferentes mini-proyectos web, cada uno implementado en **HTML/CSS/JavaScript** para mostrar funcionalidades reales.

**Meta:** Demostrar mi capacidad de crear interfaces interactivas, consumir APIs, manipular el DOM y resolver problemas prácticos con tecnologías web modernas.

---

## 🎯 Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos, animaciones y diseño responsivo
- **JavaScript (Vanilla)** - Lógica e interactividad
- **APIs externas** (según proyectos) - Integración con servicios
